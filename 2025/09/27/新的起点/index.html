# 👋 你好，这里是叶白！

欢迎来到我的个人博客！这里是我记录学习、分享思考和沉淀知识的地方。

# 📚 博客定位

# 1. 简介

yebai library 是一个以记录各种学习笔记为主的个人博客，当然后续可能会写上各种奇奇怪怪的内容。

针对个别学科采取无纸化学习策略学习效率要高于传统记笔记，因此我鼓励每个电信、计算机相关专业学生尝试去搭建一个这样属于自己的知识宝库。

# 2. 实用为主

本博客不会去追求一些过于花里胡哨的功能。

这种客观上来讲在你阅读时增加了过多的干扰项，会大大降低阅读效率，带来的体验完全是负收益。真的有人平常没事就上博客玩 live2d 吗？
